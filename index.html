Projects

SCADA Web Portal

Overview

Developed a SCADA internal web portal using ASP.NET Core Blazor Server, consisting of various automated C# scripts which generates customized term reports (SQL). After running automated scripts, data will be shown in a graph and table format, with the option of downloading file(s) to their local machine.

Features
  • User Access Report
    • All concurrent users with their respective username, name, job description, active status, and location
  • Instrumentation Summary
    • List of all instrumentations in SCADA for critical instrumentation maintenance/review
  • Data Query
    • A point's data values based on datetime given and data collection interval
  • Prover/Ticket Summary
    • Summary of a specific prover or ticket based on datetime given and station

Mechanical Turk Framework

Overview

A new system framework integrated with Amazon Mechanical Turk API (AWS), providing clients a way to conduct/manage web-based experiments (Human Intelligence Tasks), specifically around behavioural game theory. All data and experiments are structured through PostgreSQL database, with a user-friendly interface using HTML/CSS.

  • Administrators (or Requestors)
    • Consist of researchers who are responsible for managing application and creating these HITs
  • Turkers (or Workers)
    • Consist of Amazon MTurk workers who are responsible in partaking these HITs (& get paid)

Features
  • Servers/lobbies optimized for users to play (scalable) normal and extensive form games simultaneously live
  • 2-player games (i.e., TicTacToe) as an active assignment per client's request
  • Admin approval/rejection of submitted assignments, and paying bonuses to workers (option in bulk for efficiency)
  • Security payment measures to void non-approved assignments, or ones already paid

Googly

Overview

An online learning platform that serves to teach English through an immersive game, similarly to Piano Tiles, using React and Node.js. Players learn the English language by engaging in clicking/typing activities with proper automated sound pronunciations, mostly targeted for ESL learners.

Features
  • Clicking/typing of various words in the English language
  • Automated sound pronunciations
  • Speed of word animations for enhanced difficulties
  • Confetti animations

InfiniMood

Overview

A mobile mood tracking tool on Android, functioning similarly to the widely popular social networking service Twitter. This application uses Java and Android Studio, while also utilizing Firebase to manage/store new and existing user data.

Features
  • Real-time live tracking map system, similar to Snapchat's Snap Map, using Google API for users to interact simultaneously
  • Logging to unique user accounts, posting events, search/request to follow users, and customizing user profiles
  • Pop-ups of users' posts on our real-time Map system for further details and current moods per profile
